# Grok Local Agent Kit 🦾

**Functional MVP for local AI agents** - Run powerful agents offline with Ollama/LM Studio.

## Features
- Dynamic tool routing
- Multi-LLM support (Ollama, LM Studio)
- Tools: web search, file ops, shell
- Ready examples

## Install
pip install git+https://github.com/Tryboy869/grok-local-agent-kit.git

## Quickstart
```bash
grok-agent chat "Hello agent!"
```

Demos described in README.
