**Open-source toolkit for building powerful local AI agents. Ollama + MCP + autonomous capabilities. Offline-first, Grok-inspired.**

## Features
- Local LLMs via Ollama
- MCP protocol support
- Tool calling and custom tools
- Multi-agent orchestration
- CLI and Python SDK

## Quick Start

```bash
pip install git+https://github.com/Tryboy869/grok-local-agent-kit.git
ollama pull llama3.2
grok-agent chat "Your query"
```
